# Academic Mail Prompt Guide

## 한국어 학술 이메일 작성 가이드

### 수신자별 톤 매트릭스

| 수신자 | 호칭 | 문체 | 예시 |
|--------|------|------|------|
| 교수님 | OO교수님 | -습니다/-겠습니다 | "검토해 주시면 감사하겠습니다" |
| 동료 | OO님/이름 | -요/-습니다 | "확인해 주시면 감사합니다" |
| 외부 | OO님/직함 | -습니다 (비즈니스) | "회신 부탁드리겠습니다" |

### 이메일 구조

1. **인사**: 시의적절한 인사 (계절/시간대 반영)
2. **자기소개**: 첫 연락 시 소속/이름 (재연락 시 생략)
3. **본론**: bullet points → 자연스러운 문단
4. **요청/마무리**: 명확한 action item
5. **서명**: 이름 + 소속

### 프롬프트 최적화 팁

- bullet point는 구체적일수록 좋음 (날짜, 숫자 포함)
- 맥락 정보 추가 시 더 자연스러운 이메일 생성
- formal + professor 조합이 가장 높은 경어 수준
- casual + colleague 조합이 가장 편한 톤

### Upstage Solar 한국어 특성

- Solar 모델은 한국어 존칭 체계를 잘 이해
- 격식체/비격식체 전환이 자연스러움
- 학술 용어와 일상 표현의 균형이 우수
